# Diabetes Prediction with Machine Learning

This project focuses on the analysis and prediction of diabetes risk using Machine Learning techniques.  
It was developed as part of the **Samsung Innovation Campus – Artificial Intelligence program**, with the objective of applying data analysis, preprocessing, and supervised and unsupervised learning on a real-world dataset.

---

## 📌 Project Overview

The goal of this project is to:
- Analyze a large healthcare dataset
- Identify patterns related to diabetes
- Build predictive Machine Learning models
- Evaluate and compare model performance using appropriate metrics

The project covers the complete **data science pipeline**, from exploratory data analysis (EDA) to model evaluation.

---

## 📊 Dataset

- **Source:** Public dataset
- **Size:** ~100,000 records
- **Type:** Structured tabular data
- **Description:** Medical and demographic variables related to diabetes diagnosis and risk factors

---

## 🔍 Exploratory Data Analysis (EDA)

During the EDA stage, the following tasks were performed:
- Data cleaning and handling of missing values
- Statistical analysis of relevant variables
- Feature selection and transformation
- Visualization of distributions and correlations

Libraries such as **Pandas**, **NumPy**, and **Matplotlib** were used for analysis and visualization.

---

## 🤖 Machine Learning Models

Multiple Machine Learning models were trained and evaluated, including:

- Regression models
- Classification models
- **XGBoost**
- **K-Means clustering** for patient segmentation

### Techniques applied:
- Feature scaling
- Model comparison
- Hyperparameter tuning
- Performance evaluation

---

## 📈 Evaluation Metrics

Model performance was evaluated using different metrics depending on the task:

- Accuracy
- F1-score
- RMSE
- R² Score

These metrics allowed objective comparison between models and selection of the best-performing approach.

---

## 🧠 Key Results

- Identification of relevant features related to diabetes risk
- Selection of the best-performing predictive model based on evaluation metrics
- Discovery of patient groups through clustering techniques
- Practical application of Machine Learning to a healthcare-related problem
